포워드 프록시 테스트

같은 요청을 브라우저 요청과 프로그램 요청으로 각각 보내면서 포워더 동작을 비교할 수 있습니다. HTML, JSON, 리다이렉트, 쿠키, Echo, 지연 응답 시나리오를 바로 확인할 수 있습니다.

모드 안내

브라우저 모드는 X-Forwarder-ClientKind: Browser 헤더와 함께 /forwarder/api/proxy/pipe를 직접 호출합니다.

프로그램 모드는 헬퍼 API를 먼저 호출하고, 그 API가 다시 X-Forwarder-ClientKind: Program 헤더로 forwarder를 호출합니다.

세션 입력

준비되었습니다.

시나리오 프리셋

프리셋을 선택하면 요청 키, 메서드, 본문, Accept 값을 바로 채울 수 있습니다.
