graphclient module.json 편집 도구

ModuleID 모듈을 식별하는 고유 ID 입니다.
Name 모듈의 정보성 이름을 부여합니다.
IsBundledWithHost ack와 번들로 배포되는 모듈이면 true로 설정합니다.
Version 모듈 버전을 주.부.수로 관리합니다.
ModuleConfig
AuthorizationKey 비워 두면 `SystemID + Environment + HostName` 조합을 사용합니다.
BusinessServerUrl transact 모듈의 기본 실행 URL 입니다.
CircuitBreakResetSecond 로그 서버 재시도 대기 시간(초)입니다.
DefaultCommandTimeout Cypher 기본 실행 제한 시간(초)입니다.
IsContractFileWatching 계약 파일 변경 감시와 hot reload를 사용할지 여부입니다.
DefaultDataSourceID statement에 datasource가 없을 때 사용할 기본 GraphDataSourceID 입니다.
AllowClientIP 줄바꿈 또는 쉼표로 여러 IP를 입력합니다. `*`는 전체 허용입니다.
ContractBasePath
디렉토리 경로 관리
XML Cypher 계약이 위치한 루트 디렉토리 목록입니다.
IsTransactionLogging graphclient 요청/응답 로그를 남길지 여부입니다.
ModuleLogFilePath 모듈 로그 파일 경로입니다.
IsLogServer logger 모듈로 로그를 전송할지 여부입니다.
LogServerUrl logger 모듈 API 경로입니다.
IsProfileLogging Bolt 요청/응답 프로파일 로그를 남길지 여부입니다.
ProfileLogFilePath 프로파일 로그 파일 경로입니다.
EventAction
항목 관리
모듈 간 Mediator 발신 이벤트 식별자입니다.
SubscribeAction
항목 관리
모듈 간 Mediator 수신 이벤트 식별자입니다.
GraphDataSource
ApplicationID ProjectID DataSourceID GraphProvider Database 관리
`Neo4j`와 `Memgraph`만 지원하며, `bolt://` 또는 `neo4j://` URI만 허용합니다.
JSON 보기
예: `graphclient.Events.GraphClientRequest`
예: `../contracts/graphclient`
쉼표로 여러 프로젝트를 지정하거나 `*`로 전체를 허용할 수 있습니다.
예: `bolt://localhost:7687`, `neo4j://cluster:7687`
Neo4j는 명시적 DB 선택에 사용하고, Memgraph는 보통 `memgraph` 값을 둡니다.
연결 문자열, 사용자명, 비밀번호, 데이터베이스 값이 암호화돼 있으면 체크합니다.
