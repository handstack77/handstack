1) 연결 설정

서버 주소와 관리 헤더를 먼저 입력한 뒤 키 검증으로 연결 상태를 확인하세요.

2) 대상(Targets) 제어

대상 목록 조회 후 대상 ID를 기준으로 상태 조회, 시작/중지/재시작을 실행합니다.

3) 설정 API

설정 대상 ID를 입력하고 상태를 조회하거나 JSON 설정 값을 저장합니다.

4) 모듈 API

모듈 ID와 선택 대상 ID를 입력해 모듈 설정을 조회 또는 저장합니다.

5) 모니터링/로그

로그 파일명은 비워두면 최신 파일을 사용합니다. 경로 형식(module/file.log)도 입력할 수 있습니다.
